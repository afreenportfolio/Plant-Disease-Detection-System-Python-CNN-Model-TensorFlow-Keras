# Plant Disease Detection System

![Type](https://img.shields.io/badge/Type-AICTE_Internship_Project-purple) 
![Status](https://img.shields.io/badge/Status-Completed-green)

This project employs convolutional neural networks (CNN) to identify diseases in plant images, aiding farmers and researchers in early detection. The model processes images, classifies diseases, and provides insights for better crop management.

## Features

1. Image classification using trained CNN models.
2. Data preprocessing and augmentation for improved accuracy.
3. Hyperparameter tuning and evaluation metrics.
4. Automated validation scripts for consistency.

## Technologies Used

1. Framework: TensorFlow-Keras
2. Language: Python
3. Tools: Google Jupyter Notebooks

## Installation

- Clone the repository: `git clone https://github.com/afreenportfolio/plant-disease-detection-system.git`
- Install dependencies: `pip install tensorflow keras numpy matplotlib`
- Download dataset: [Dataset](https://drive.google.com/file/d/12OS3a0bLn1zDnhqv5KRRH_BfFml6gCVX/view?usp=drive_link)

## Usage

- Train the model: Download [Code](https://github.com/afreenportfolio/plant-disease-detection-system/blob/main/Plant_Disease_Detection_SystemC6_ipynb_txt.ipynb) and open it in Google Collab & run it
- View results and accuracy reports.
- Try to increase the accuracy from **89%**!
